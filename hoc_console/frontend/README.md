# HOC Frontend

React + Vite + TypeScript dashboard scaffold.

## Setup

```bash
cd hoc_console/frontend
npm install
npm run dev
```

Connects to `ws://localhost:8765`. See [04-hoc-console-design.md](../../docs/design/04-hoc-console-design.md).

## Status

Scaffold only — UI components to be implemented in M5 milestone.
