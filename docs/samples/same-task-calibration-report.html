1. 实验目的

解决跨源对比中任务不一致（bridge 正弦 demo vs LeRobot pick_and_lift）导致的指标解释困难。 本实验在同一 JointTrajectory 命令下，对比 bridge 双源模式中的 Sim（理想 PyBullet）与 Real（带 domain randomization 的 PyBullet），使 KL / W1 / MMD 具有可解释的物理含义。
